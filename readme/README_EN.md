# screen-capture-suite-tracker

Bug tracking, feature feedback, and project information management repository for a **screen capture suite** built on ComfyUI nodes and companion software plugins (e.g., Blender).

---

## Product Introduction

A foundational control tool built specifically for AIGC image and video creation. Breaking through the limitations of a single director's console, it provides both general and customized paths, offering comprehensive control over AIGC creation including pose, first/last frame, multi-grid, video, composition, lighting, and storyboard prompt reverse engineering — a practical assistant for professional-level creation.

### Addressing Pain Points

The current market offers a wide variety of director consoles, but limited by functional diversity and controllability, a single tool often cannot meet the complex and ever-changing needs of AIGC creation. This suite breaks down software barriers, helping you flexibly build a personalized director console.

### Core Features

**General Path: Free Control, Flexible Recording**
- Custom recording area: Freely create, move, and adjust resolution to precisely capture images and image sequences from any area on the screen.
- Background silent recording: While capturing images, the suite automatically and silently records operation trajectories in the background — ideal for advanced users who need review or model distillation.

**Custom Path: Dedicated Optimization, Streamlined Workflow**
- Dedicated software adaptation: Fixed recording areas with deep customization for specific software (e.g., Blender).
- Full feature inheritance: Retains image/sequence capture and background silent recording capabilities, ready to use out of the box.

**Sequence Frame Cropping: Precise Output**
- Built-in professional cropping nodes that can precisely crop long captured sequences into multiple segments and output them directly, seamlessly connecting to subsequent workflows.

### Suite Contents

- **ComfyUI Node Plugin**: Contains 2 core nodes (Screen Capture Node, Sequence Frame Crop Node).
- **Blender Plugin**: A Blender-optimized version of the Screen Capture Node for efficient camera and 3D viewport capture.

### Usage Suggestions

- **High-resolution screens**: Use split-screen functionality.
- **Low-resolution screens**: Connect an external monitor to separate the capture setup from the operation area for improved workflow efficiency.
