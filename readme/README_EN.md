Bug tracking, feature feedback, and project information management repository for a **screen capture suite** built on ComfyUI nodes and companion software plugins (e.g., Blender).

## Bug Reports & Feature Requests

If you encounter any issues during use, you can provide feedback in this repository.

### Issue Submission Guidelines

When submitting an Issue, please include:
- ComfyUI version
- Specific software version (e.g., Blender 4.2)
- Operating system
- Description of the problem
- Screenshots of the issue
