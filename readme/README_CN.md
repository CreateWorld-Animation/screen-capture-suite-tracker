# screen-capture-suite-tracker

本仓库是一个基于 **ComfyUI 节点** 和特定软件插件（如 Blender）配套的 **屏幕捕捉套件** 的 Bug 追踪、功能反馈与项目信息管理仓库。

---

## 产品简介

专为 AIGC 图像与视频创作打造的底层控制工具。突破单一导演台的局限，提供通用与定制双路径，为 AIGC 创作提供姿态、首尾帧、多宫格、视频、构图、灯光、分镜提示词反推等全方位控制，是实现专业级创作的实用辅助工具。

### 解决痛点

当前市面上的导演台五花八门，但受限于功能多样性和可控性，单一工具往往无法满足 AIGC 创作中复杂多变的需求。本套件打破软件壁垒，协助您灵活搭建个性化导演台。

### 核心功能

**通用路径：自由掌控，灵活录制**
- 自定义录制区域：可自由创建、移动并调整分辨率，精准捕获屏幕上任意区域的图像与序列帧。
- 后台静默记录：在捕获图像的同时，后台自动静默记录录制区域的操作轨迹，适合需要复盘或模型蒸馏的高级用户。

**定制路径：专属优化，固化流程**
- 专属软件适配：录制区域固化，针对特定软件（如 Blender）进行深度定制优化。
- 全功能继承：同样具备图像/序列帧捕获及后台静默记录操作的功能，实现开箱即用。

**序列帧裁剪：精准输出**
- 内置专业裁剪节点，可将捕获的长序列帧进行多段精准裁剪并直接输出，无缝衔接后续工作流。

### 套件包含内容

- **ComfyUI节点插件**：内含 2 个核心节点（屏幕捕获节点、序列帧裁剪节点）。
- **Blender专属插件**：作为屏幕捕获节点的 Blender 定制优化版，实现相机和 3D 视口区域的高效捕获。

### 使用建议

- **大分辨率屏幕**：建议使用分屏功能。
- **小分辨率屏幕**：建议外接扩展屏，将捕获设置与操作区域分离，提升工作流效率。
